# nixtropic
Tropic square's [libtropic-util](https://github.com/tropicsquare/libtropic-util), for Nix

To build:
  - nix build github:jjacke13/nixtropic#libtropic-util_TS1302
    
or:

  - nix build github:jjacke13/nixtropic#libtropic-util_TS1301
